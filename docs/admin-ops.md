## Admin 配置管理系统操作手册

### 入口
- 管理后台：`/admin-system`
- 需要先输入 `ADMIN_TOKEN` 才能加载数据

### 三级联动配置
1. 选择月份
2. 选择舞室
3. 选择分店
4. 进入“爬虫实例”页进行单个重启或一键重启

### 图片结果
- 支持分页浏览、缩放预览、下载

### 解析结果
- 表格展示，支持导出 CSV

### 数据监控
- 点击统计卡片可查看失败列表

### 规则管理
- 新增规则会自动生成版本号
- 点击“设为当前”进行版本回滚

### 备份与恢复
- 备份会导出 configs/templates/rules
- 恢复会覆盖同 ID 数据

### 中英文切换
- 顶部语言选择器可切换界面语言
