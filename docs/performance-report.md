- 评估单次任务平均耗时与失败率。

### 2. 测试环境
- 前端：Railway 部署（admin-frontend）
- 后端：Railway 部署（admin-backend）

- 数据库：Supabase（Postgres）

### 3. 测试方法
1. 在 admin-backend 部署环境中执行批量触发：
   - 调用 `/api/configs/run-all` 或脚本对 `/api/crawler/crawl` 发起并发请求。
2. 每批 10/50/100 条配置，记录：
   - 平均耗时（每批整体完成耗时 / 任务数）
   - OCR 成功率

3. 观察 Dashboard 中 need_manual_upload 告警触发情况。
